## 🛠️ How to Run?

```bash
# Clone this repository
git clone <repo-link>

# Navigate to project folder
cd my-react-tutorial

# Install dependencies
npm install

# Run the app
npm start
```

Open your browser at `http://localhost:3000` 🌐 and see the magic!
